## 已完成（截至 2026-07-03）

### 核心架構
- **Supabase Storage** 儲存每日快照 JSON（`latest.json`）
- **GitHub Actions Cron**（每日 ~19:45 台北時間）自動抓盤後資料
- **Vercel** 前端部署，URL：https://franky-stock-dashboard.vercel.app/

### 板塊泡泡圖
- **座標軸定義**（與 tide-tw.app 一致）
  - X 軸：近 5 日三大法人均淨買超（千張，右=買超/左=賣超）
  - Y 軸：加速指標 = (近5日均/近20日均) - 1（上=加速/下=放緩）
  - 泡泡大小：近 5 日買賣均量（千張）
- **四象限**
  - 右上（TR）= 漲潮：資金加速流入
  - 左上（TL）= 觀望：資金流出但放緩
  - 左下（BL）= 退潮：資金流出加速
  - 右下（BR）= 輪動：資金流入但放緩
- **互動功能**
  - 象限篩選按鈕（觀望/漲潮/退潮/輪動）
  - 點擊象限放大
  - Pinch-to-zoom（手機）+ 滾輪縮放（桌面）+ 拖曳平移
  - 重置按鈕（縮放後出現）
  - 點擊泡泡 → 開啟個股面板（底部 Sheet）

### 個股面板（SectorPanel）
- 每支個股：**單行可橫向拖曳**
  - 代號 | 名稱 | ▲漲跌% | 合計±X 外資±X 投信±X 自營±X 距高 距低
- 依細分產業分組顯示
- 距高/低%：依照用戶設定的 N 天計算

### T86 三大法人資料
- **Bug 修正**：原本 `selectType=ALL` 無小計，改為依 `stockMap.sector` 手動分組
- 外資 = row[4] + row[7]，投信 = row[10]，自營 = row[11]（合計），三大合計另計
- 1080 支個股已補全 sector 欄位（seed-sectors.mjs）

### 大盤 K 線
- 收盤最高/最低標記線改用收盤價（與 MarketSignalCards 一致）

## 技術決策備忘

| 項目 | 決策 |
|------|------|
| 盤後資料 | TWSE 官方 API（免費，無限制） |
| 三大法人 | T86 API，selectType=ALL，手動依 sector 分組 |
| 個股 OHLC | Supabase Postgres（非 Storage，避免 5MB 限制） |
| 距高/低% | 前端計算（N 值由 user 決定，不固定） |
| 顏色慣例 | 紅=漲/利多，綠=跌/利空（台灣習慣） |
| 快照欄位 | closes[]/dates[] 暫時保留；OHLC table 建好後可移除（省 ~4.5MB） |
