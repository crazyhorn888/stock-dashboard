## 更新時序總覽

```
15:30       FMTQIK 就緒（Guard 2 開關）→ 大盤 K 線可更新
16:00–17:00 n8n Phase 1 抓三大法人 / 期貨 / VIX / PCR
              → chips JSON 寫進 Supabase chips/{date}.json
              → dispatch GitHub Actions
17:00–22:00 TWSE MI_MARGN 融資餘額就緒（同一次 cron 順帶抓）
16:00–22:00 TWSE STOCK_DAY_ALL 個股股價（偶有深夜，加開 23:30 cron 補捉）
18:00 起    GitHub Actions cron（18:00 / 18:30 / 19:00 / 20:00 / 21:00 / 22:00 / 23:30）
              → 抓 TWSE T86 板塊三大法人
              → calcSectors() 計算泡泡圖
              → 上傳 latest.json → Supabase
              → calc-signals.mjs → signals → Firestore
```

TPEX 三大法人 ─────────────────────────────→ stocks[].foreignNetBuy（上櫃）

TWSE MI_MARGN ─────────────────────────────→ indexHistory[today].chips.margin_amount
n8n Phase 1 → Supabase chips/ ────────────→ indexHistory[today].chips（外資/投信/自營/期貨/VIX/PCR）
